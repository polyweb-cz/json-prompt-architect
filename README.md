## What is this?

JSON Prompt Architect is a visual tool that helps you create strict JSON schemas and forms to guide Artificial Intelligence (like ChatGPT, Gemini, Claude) to output structured data exactly how you need it.

Instead of writing complex JSON manually, you build the structure visually, set validation rules (like minimum/maximum length), and then fill out a user-friendly form to generate the final JSON prompt.

## How to Use

1.  **Edit Mode:** Define your structure. Add text fields, long text areas, or organize them into sections. Set limits (e.g., "Max 64 chars") to keep data concise.
2.  **Use Mode:** Switch tabs to see the generated form. Fill in the values. The tool validates your input in real-time.
3.  **Output:** Copy the resulting valid JSON or click **"Open in ChatGPT"** to start a conversation with the AI using your structured data immediately.
